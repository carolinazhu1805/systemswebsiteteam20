Project background and Partner Introduction

Our project extends the original Illumiroom project from precious years by adding a main User Interface which allows users to switch between the immersive graphics that Illumiroom can provide or the web app version of the program. Illumiroom is originally a program which allows users to have more immersive experiences by connecting to a projector. Project Goals

Our goal is to develop an intuitive User Interface that allows users to seamlessly switch between the Windows app version and the Immersive Graphics Mode. Furthermore our program should facilitate the integration of user-defined triggers within the IllumiRoom framework, ensuring these triggers effectively initiate specific projections. This ensures a seamless and dynamic user experience, tailored to activate immersive projections based on predefined criteria, enriching the overall interaction with the IllumiRoom environment.
